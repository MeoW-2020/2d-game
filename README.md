## Для запуска игры:
- скачать и распаковать архив `Game`
- запустить файл `In search of a Sapphire.exe`


## Управление
идти - `WASD`/`стрелки`  
атака - `пробел`  
взаимодействовать (при появлении вопр. знака над игроком) - `F`  
пауза - `esc`
